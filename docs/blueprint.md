# **App Name**: ClaimHero

## Core Features:

- Navbar: Sticky navbar with ClaimHero logo and right-side links: Home, Features, Guide, Contact. Mobile responsive with collapsible menu.
- Hero Section: Hero section with large bold tagline: 'Small Claims. Big Help,' subheadline, and CTA button that scrolls to intake form.
- Smart Eligibility Intake Form: Smart Eligibility Intake Form: Checks if amount < $10,000, retrieves court info based on zip code, and validates the incident date. Displays eligibility status and recommended next steps.
- Demand Letter Generator: Demand Letter Generator: Generates a legal-style demand letter from form inputs. Outputs letter in a modal with 'Copy' and 'Print' buttons.
- Claim Strength Analyzer: Claim Strength Analyzer: Assesses the strength of the claim based on user inputs. Outputs verdict with explanation.
- Step-by-Step Guide: Step-by-Step Guide (Checklist): Interactive checklist to guide users through the small claims process. Uses localStorage to save progress.
- Court Finder: Court Finder: Takes zip code and returns court information.

## Style Guidelines:

- Primary color: Saturated blue (#2962FF), evoking trust and reliability.
- Background color: Light blue (#E5EFFF), a very desaturated variant of the primary hue to promote calm and readability.
- Accent color: Yellow-green (#B0E050) to signal calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Responsive design for mobile, tablet, and desktop. Smooth scroll and subtle animations.
- Use clear, professional icons related to legal processes.
- Subtle animations to enhance user experience and provide feedback.
